# Tokens

APT tokens define all design primitives for color, spacing, typography, and elevation. Tokens are used via Tailwind classes and config package exports.

## Examples
- `bg-background`, `text-foreground`, `border-border`, `bg-card`, `text-card-foreground`
- Spacing: `p-4`, `m-2`, etc.
- Typography: `font-bold`, `text-lg`, etc.
