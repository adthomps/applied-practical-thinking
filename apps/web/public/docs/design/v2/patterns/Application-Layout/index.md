## Intent

Define layout scaffolding for multi-screen applications, including navigation, sidebars, and responsive behavior.

## Anatomy

- Top navigation
- Side navigation (optional)
- Content region
- Footer / utility area

## Accessibility

Ensure landmark roles, skip links, and keyboard focus order are correct across layout regions.

## Tokens & Theming

Use tokens for layout breakpoints, container widths, and gutter spacing.
