# APT Validation Reports Directory

This folder stores portfolio-level outputs generated by APT validation and sweep scripts.

## What Is Here

- `project-profile-validation-sweep-YYYY-MM-DD.json`: machine-readable workspace summary.
- `project-profile-validation-sweep-YYYY-MM-DD.md`: human-readable dashboard summary.

## How To Generate

From `apt-principles`:

```bash
npm run run-all-checks
npm run sweep:project-profiles
```

Use `npm run run-all-checks` when you want one operator command that runs canonical validation, sweep generation, and selected repo quality checks before printing a compact pass/fail summary.

Optional:

```bash
npm run sweep:project-profiles -- --date YYYY-MM-DD
npm run sweep:project-profiles -- --repos repo-a,repo-b
npm run sweep:project-profiles -- --workspace-root ../
```

## How To Use The Results

1. Read the markdown report first to triage by severity, APT layer, and action queue.
2. Open the JSON report to automate follow-up (issue creation, dashboards, trend checks).
3. For any repo-specific deep dive, open that repo's local static report at `docs/apt/reports/static/project-profile-validation-sweep-YYYY-MM-DD.json`.
