## Overview
shadcn UI Builder is a visual interface for configuring and generating UI projects. It enables developers to customize themes, components, and project setup before writing code, promoting open code ownership and flexibility.

### Features
- Real-time customization: themes, colors, fonts, border radius, icon sets
- Component-level preview and shuffle
- Supports Next.js, Vite, TanStack Start, v0 frameworks
- Choice of component foundations (Radix UI, Base UI)
- Generates custom `shadcn create` command
- Open code: components are copied into your project

### Use Cases
- Rapid prototyping and design system setup
- Enterprise teams needing full UI code control
- Experimenting with visual styles
- Custom theming and component flexibility

## Integration
This system can be used to bootstrap new projects or enhance existing design systems. It aligns with APT's philosophy of open, customizable UI components.

## Example
To start a new project:
1. Visit [shadcn UI Builder](https://ui.shadcn.com/create)
2. Configure your preferences
3. Run the generated command in your project
