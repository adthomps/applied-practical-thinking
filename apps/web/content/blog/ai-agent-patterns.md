## Introduction

AI agents are everywhere, but most fall into a handful of patterns. This post surveys the most common ones and when to use each.

## 1. Single-Turn Chatbot

- Stateless
- Good for Q&A, search, simple tasks

## 2. Multi-Turn Assistant

- Maintains conversation history
- Handles context, follow-ups

## 3. Tool-Using Agent

- Can call APIs, run code, fetch data
- Useful for automation, research

## 4. Workflow Orchestrator

- Coordinates multiple agents or steps
- Good for complex, multi-stage tasks

## Choosing a Pattern

- Start simple, add complexity as needed
- Consider user expectations and risk

## Conclusion

Patterns help you avoid reinventing the wheel. Pick the right one for your use case, and evolve as you learn.
