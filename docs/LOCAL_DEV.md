# Local Development

- Start frontend: `pnpm --filter "./apps/web" dev`
- Start backend: `pnpm --filter "./apps/worker" dev`
- Use two terminals for local dev
- Use Vite proxy for /api to backend
- Install all dependencies at the repo root with `pnpm install`
- All contributors must use [pnpm](https://pnpm.io/) (see README for details)
