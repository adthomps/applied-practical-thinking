# AI_PROMPT.md

## AI Prompt Authoring

- All prompts live in `apps/web/ai/prompts/`
- Use markdown for prompt files
- Do not hardcode prompt strings in code
- Each prompt should have an owner and clear intent
- Prompt changes must be reviewed and documented when they affect behavior
